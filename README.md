# Proyecto ETL - Análisis de Opiniones

Este proyecto implementa un pipeline ETL en Python que carga datos desde archivos CSV
a una base de datos SQL Server.

## Tecnologías
- Python
- Pandas
- SQLAlchemy
- SQL Server
- Visual Studio 2022

## Flujo ETL
1. Extracción de datos desde archivos CSV
2. Limpieza y normalización de datos
3. Carga de datos en la base de datos

Nombre: pruebin
